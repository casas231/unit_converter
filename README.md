# Chuyển đổi đơn vị
Trang web đơn giản để chuyển đổi đơn vị.
Ý tưởng từ bài [Unit Converter](https://roadmap.sh/projects/unit-converter) trên nền tảng [roadmap.sh](https://roadmap.sh).

## Tính năng
- Độ dài: km, mi.
- Khối lượng: kg, lb.
- Nhiệt độ: °C, °F.
